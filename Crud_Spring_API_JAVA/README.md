# Meeting room manager

## Stack used:


 * Spring Web
 * SpringData JPA
 * H2 Database
 * Java 8
 * Maven
 
## Endpoints created in the API

* Create meeting room
POST - /api/v1/rooms

* List all rooms
GET - /api/v1/rooms

* Search for a room by ID
GET - /api/v1/rooms/{id}

* Update a room by Id
PUT - /api/v1/rooms/{id}
